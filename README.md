# Protocol-Dependent Legitimacy of Global Training Metrics

Minimal computational experiment exploring when global training metrics remain stable under different optimization protocols in multi-objective machine learning systems.

Three training protocols are tested:

- simultaneous optimization
- sequential optimization A → B
- sequential optimization B → A

The experiment shows that when objectives are compatible, global metrics remain stable across protocols. When objectives are structurally incompatible, protocol dependence emerges and global closure cannot be achieved.
